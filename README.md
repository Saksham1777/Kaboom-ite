An adrenaline-pumping, retro-inspired space shooter where you blast rocks, grab power-ups, and try not to go Kaboom!

Kaboom-ite is a fast-paced survival game built with Python and Pygame. Take control of your ship, navigate through a dense asteroid field, and survive as long as possible. The longer you stay alive, the more intense the chaos becomes!

## 🌟 Features

- **Dynamic Difficulty**: Asteroids spawn more frequently and move faster over time
- **Power-Up System**: 
  - **Penetration**: Bullets tear through multiple asteroids
  - **Shield**: Become briefly invincible and smash through asteroids
- **Fluid Controls**: Smooth physics-based movement with friction and mouse-aimed precision
- **Immersive Audio**: Satisfying blasts, firing sounds, and "bruh" moments on death
- **Screen Wrapping**: Asteroids seamlessly wrap around screen boundaries

## 🎮 How to Play

- **Launch**: Press ENTER to start/restart
- **Move**: W, A, S, D keys
- **Aim**: Point your cursor where you want to fire
- **Shoot**: Left-click to blast asteroids
- **Survive**: Avoid collisions (unless shielded)!

## 💻 Technical Details

- **Language**: Python
- **Engine**: Pygame
- **Math**: Numpy for backend calculations
- **Architecture**: Object-Oriented design with dedicated modules for game logic, models, and utilities
